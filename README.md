
## 主要功能 

快速创建项目，初始化项目结构及配置


## 可用命令

```bash
# 添加模版
$ yunke add template-rep 

# 查看本地模版
$ yunke list 

# 初始化项目
$ yunke init template-rep or template-name

# 删除本地模版
$ yunke remove template-name

```


## 快速开始

```bash
# 全局安装
$ npm i -g yunke-cli

# 切换到工作目录
$ cd path/to/workspace

# 初始化项目
$ yunke init https://github.com/fbi-templates/fbi-project-vue.git my-project

# 切换到项目目录
$ cd my-project

```
